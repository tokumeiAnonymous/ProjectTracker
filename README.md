# ProjectTracker
Simple Web App with CRUD and Security

![](https://github.com/tokumeiAnonymous/ProjectTracker/blob/main/Features.gif)

## Description
This Project is used to track changes in the project that your building.

Technology used:
  - Spring Boot
  - Spring Web
  - Spring MVC
  - Spring Security
  - Spring JPA
  - Java Language
  - Bootstrap 5
  - Thymeleaf
  - MySQL
  - Hibernate
